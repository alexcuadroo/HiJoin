# Changelog

## [1.0.0] - 2026-01-01

### Added
- Lanzamiento inicial
- Soporte de títulos personalizables al conectarse
- Soporte de múltiples colores y estilos
- Configuración flexible con `config.yml`
- Comando `/hijoin reload` para recargar configuración
- Comandos `/hijoin` e `/hijoin info` para información
- Sistema de permisos con `hijoin.admin`
