# Changelog

## [1.1.0] - 2026-01-01

### Added
- Caché de colores para optimizar búsquedas repetidas
- Validación de permisos en comando `/hijoin reload`
- Documentación completa en README

### Changed
- Actualizada API a Paper 1.21
- Mensajes de consola deshabilitados por defecto para ahorrar CPU
- Mejoras en validación de listas vacías

### Optimized
- Reducción significativa de uso de CPU con caché de colores
- Mejor manejo de iteraciones en estilos de texto

## [1.0.0] - 2026-01-01

### Added
- Lanzamiento inicial
- Soporte de títulos personalizables al conectarse
- Soporte de múltiples colores y estilos
- Configuración flexible con `config.yml`
- Comando `/hijoin reload` para recargar configuración
- Comandos `/hijoin` e `/hijoin info` para información
- Sistema de permisos con `hijoin.admin`
